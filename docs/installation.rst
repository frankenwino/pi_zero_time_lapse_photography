============
Installation
============


Stable release
--------------

To install Pi Zero Time Lapse Photography, run this command in your terminal:

.. code-block:: console

    $ pip install pi_zero_time_lapse_photography

This is the preferred method to install Pi Zero Time Lapse Photography, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Pi Zero Time Lapse Photography can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/frankenwino/pi_zero_time_lapse_photography

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/frankenwino/pi_zero_time_lapse_photography/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/frankenwino/pi_zero_time_lapse_photography
.. _tarball: https://github.com/frankenwino/pi_zero_time_lapse_photography/tarball/master
